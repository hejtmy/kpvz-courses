Seznam zdrojů pro samostudium PsychoPy.

## Oficiální dokumentace

### PsychoPy Documentation
- **URL:** [https://www.psychopy.org/documentation.html](https://www.psychopy.org/documentation.html)
- **Obsah:** Kompletní oficiální dokumentace, API reference
- **Vhodné pro:** Všechny úrovně, především jako referenční materiál

### PsychoPy Builder Tutorial
- **URL:** [https://www.psychopy.org/builder/index.html](https://www.psychopy.org/builder/index.html)
- **Obsah:** Tutoriály pro Builder interface
- **Vhodné pro:** Začátečníky, vizuální tvorba experimentů

### PsychoPy Coder Tutorial
- **URL:** [https://www.psychopy.org/coder/index.html](https://www.psychopy.org/coder/index.html)
- **Obsah:** Programování experimentů v Pythonu
- **Vhodné pro:** Pokročilé uživatele s Python znalostmi

## Video tutoriály

### Jon Peirce's YouTube Channel
- **URL:** [https://www.youtube.com/@jonpeirce](https://www.youtube.com/@jonpeirce)
- **Obsah:** Video tutoriály od tvůrce PsychoPy
- **Vhodné pro:** Vizuální learners, step-by-step návody

### PsychoPy Tutorials by Damien Mannion
- **URL:** [https://www.djmannion.net/psych_programming/vision/intro/intro.html](https://www.djmannion.net/psych_programming/vision/intro/intro.html)
- **Obsah:** Detailní tutoriály s kódem
- **Vhodné pro:** Středně pokročilé, programming-oriented

## Online kurzy

### Creating Experiments in PsychoPy (Pavlovia)
- **URL:** [https://pavlovia.org/docs/home/about](https://pavlovia.org/docs/home/about)
- **Obsah:** Online experimenty, Pavlovia platforma
- **Vhodné pro:** Online výzkum, vzdálený sběr dat

## Knihy a publikace

### Building Experiments in PsychoPy (Peirce & MacAskill)
- **ISBN:** 978-1473991392
- **Obsah:** Comprehensive guide od autorů PsychoPy
- **Vhodné pro:** Hloubková znalost, reference

## Fóra a komunita

### PsychoPy Discourse Forum
- **URL:** [https://discourse.psychopy.org/](https://discourse.psychopy.org/)
- **Obsah:** Komunitní podpora, Q&A
- **Vhodné pro:** Řešení problémů, best practices

### GitHub Issues
- **URL:** [https://github.com/psychopy/psychopy/issues](https://github.com/psychopy/psychopy/issues)
- **Obsah:** Bug reports, feature requests
- **Vhodné pro:** Technické problémy

## Příklady experimentů

### PsychoPy Demos
- **Lokace:** Součást PsychoPy instalace (Menu > Demos)
- **Obsah:** Předpřipravené experimenty
- **Kategorie:**
  - Stroop task
  - Visual search
  - N-back task
  - Posner cueing
  - Motion perception

### Open Science Framework (OSF)
- **URL:** [https://osf.io/](https://osf.io/)
- **Obsah:** Sdílené experimenty a materiály
- **Hledání:** "PsychoPy experiment"
